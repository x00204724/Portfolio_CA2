Bootstrap Features Used in Portfolio

Hosted URL: https://github.com/x00204724/Portfolio_CA2

1. Layout Components

  • container - Main content wrapper with responsive padding
  • row - Grid row for column layout
  • col-md-6, col-md-4 - Responsive grid columns

2. Navigation Components

Navbar

  • navbar, navbar-expand-lg - Responsive navigation bar
  • navbar-dark, bg-dark - Dark theme styling
  • navbar-brand - Brand/logo area

Breadcrumb

  • breadcrumb, breadcrumb-item - Navigation breadcrumbs

3. Content Components

Cards

  • card, card-body - Content containers
  • card-title, card-text - Card content elements
  • h-100 - Full height cards

Accordion

  • accordion, accordion-item - Collapsible content sections
  • accordion-header, accordion-button - Accordion controls
  • accordion-collapse, accordion-body - Collapsible content
  • collapse, show, collapsed - Collapse states
  • Used in Education and FAQ sections

List Groups

  • list-group, list-group-item - Styled lists
  • list-group-item-action - Interactive list items

4. Interactive Components

Modal

  • modal, fade - Modal dialog with animation
  • modal-dialog, modal-content - Modal structure
  • modal-header, modal-title, modal-body, modal-footer - Modal sections
  • data-bs-toggle="modal", data-bs-target - Modal triggers
  • data-bs-dismiss="modal" - Modal close functionality
  • Used in Contact section

Offcanvas

  • offcanvas, offcanvas-end - Slide-in sidebar
  • offcanvas-header, offcanvas-title, offcanvas-body - Offcanvas structure
  • btn-close - Close button
  • data-bs-toggle="offcanvas" - Offcanvas trigger
  • data-bs-dismiss="offcanvas" - Offcanvas close
  • Used for Quick Links navigation

5. Form Components

  • form-control - Styled form inputs
  • form-label - Form field labels
  • form-text - Helper text
  • invalid-feedback - Validation error messages
  • Used in Contact Modal

6. Button Components

  • btn - Base button class
  • btn-primary - Primary action buttons
  • btn-outline-primary - Outlined primary buttons
  • btn-outline-light - Outlined light buttons
  • btn-outline-dark - Outlined dark buttons
  • btn-secondary - Secondary buttons

7. Utility Components

Badges

  • badge, bg-primary - Skill tags and labels

Spinner

  • spinner-border - Loading indicator
  • Used in Contact section

8. Utility Classes

Spacing

  • py-5, py-3, py-2 - Vertical padding
  • mb-3, mb-2, mb-0 - Margin bottom
  • mt-3 - Margin top
  • ms-2 - Margin start (left)
  • p-3, p-4 - All-around padding

Flexbox

  • d-flex - Flexbox display
  • align-items-center - Vertical alignment
  • justify-content-between - Space between items

Display

  • d-none - Hide element

Text

  • text-center - Center text alignment
  • text-danger - Danger/error text color
  • text-muted - Muted text color
  • lead - Larger text for emphasis

Background

  • bg-light - Light background color
  • bg-dark - Dark background color

9. JavaScript Attributes

  • data-bs-toggle - Trigger Bootstrap components (collapse, modal, offcanvas)
  • data-bs-target - Specify target element ID
  • data-bs-parent - Accordion parent for exclusive collapse
  • data-bs-dismiss - Close/dismiss components
  • data-bs-theme - Theme variant (dark/light)

10. Responsive Features

  • navbar-expand-lg - Responsive navbar breakpoint
  • col-md-6, col-md-4 - Responsive grid columns
  • Mobile-first responsive design throughout

Summary

Total Bootstrap Components Used: 10+ major components

Total Bootstrap Classes Used: 60+ utility and component classes

JavaScript Features: Modal, Offcanvas, Accordion, Collapse
